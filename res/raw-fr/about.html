À propos

DroidFish est une interface graphique d'échecs riche en fonctionnalités, associée au très fort module Stockfish.

Fonctionnalités

  • Livre d'ouverture
  • Reprise des coups illimitée
  • Copier/coller depuis/vers le presse-papier
  • Mode analyse
  • Mode deux joueurs
  • Mode ordinateur vs ordinateur
  • Configuration de positions
  • Support complet des PGN
  • Niveau de jeu réglable
  • Modules UCI tiers
  • Tables de finales Gaviota

Conseils

  • Faites un appui long sur les boutons mode/gauche/droite pour d'avantage de commandes.
  • Lors de la navigation dans une partie, mettre le mode Modifier/rejouer la partie vous permet la reprise des demi-coups.

Livres d'ouverture

DroidFish inclut un petit livre d'ouverture. Pour en utiliser un plus complet, vous pouvez configurer DroidFish pour utiliser des livres externes polyglot ou CTG:

  1. Copiez un ou plusieurs livres polyglot vers le répertoire DroidFish de la carte SD. Les livres polyglot doivent avoir l'extension de fichier .bin. Note! La gestion des fichiers par Android est sensible aux majuscules, d'où l'extension qui ne doit être ni .Bin ni .BIN, mais .bin.
  2. Copiez un ou plusieurs fichiers CTG vers le répertoire DroidFish. Un livre CTG est composé de trois fichiers dont les extensions sont .ctg, .ctb et .cto. Vous devez copier ces trois fichiers.
  3. A partir de menu du programme, activez la fonction Sélectionner le livre d'ouverture.
  4. Sélectionnez le livre d'ouverture que vous voulez utiliser.

Fichiers PGN

DroidFish peut charger des parties depuis des fichiers PGN.

  1. Copiez un ou plusieurs fichiers PGN dans le répertoire DroidFish/pgn de la carte SD.
  2. Faites un appui long sur l'échiquier et sélectionnez Charger une partie PGN.
  3. Sélectionnez le fichier, puis la partie à charger.

Pour modifier l'en-tête de la partie, tels que les noms des joueurs, faites un appui long sur la liste des coups et sélectionnez Modifier l'en-tête. Pour modifier les commentaires d'un coup, faites un appui long sur ce coup et sélectionnez Modifier les commentaires.

Pour sauvegarder ou modifier une partie, faites un appui long sur l'échiquier et sélectionnez Sauvegarder la partie au format PGN. Sélectionnez le fichier à sauvegarder ainsi que son emplacement.

Note! Si l'application OI File Manager est installée, n'importe quel répertoire pourra être utilisé pour charger/sauvegarder des parties.

Pour ajouter l'analyse de l'ordinateur aux variantes de la partie, faites un appui long sur les informations d'analyse quand l'ordinateur réfléchit, et sélectionnez Ajouter l'analyse. La variation principale est ajoutée à la liste des coups de la partie.

En mode analyse, il est possible d'ajouter un coup nul. Sachez cependant que les coups nuls ne sont pas inclus dans le standard PGN et pourraient de ce fait ne pas fonctionner avec tous les lecteurs PGN.

Fichiers scid

Si vous avez Scid on the go installé, vous pouvez lire des données Scid directement depuis DroidFish. Faites un appui long sur l'échiquier et sélectionnez Charger une partie Scid.

Fichiers FEN/EPD

DroidFish peut charger des positions à partir de fichiers FEN/EPD.

  1. Copiez un ou plusieurs fichiers FEN/EPD dans le répertoire DroidFish/epd de la carte SD.
  2. Faites un appui long sur l'échiquier et sélectionnez Charger une position depuis un fichier.
  3. Sélectionnez le fichier, puis la position dans celui-ci.

Vous pouvez également charger des positions à partir de programmes tiers, tels que ChessOcr (payant), qui mermet de prendre une photo d'un diagramme d'échecs et de le retranscrire automatiquement dans DroidFish.

  1. Faites un appui long sur l'échiquier et sélectionnez Récupérer la position.
  2. Suivez les instructions du programme tiers.

Modules UCI

DroidFish peut exploiter les modules UCI tiers.

  1. Copiez un ou plusieurs fichiers binaires des modules UCI dans le répertoire DroidFish/uci de la carte SD. Note! Les fichiers binaires doivent être compilés pour Android.
  2. Depuis le menu du programme, choisissez Gérer les modules d'échecs et Sélectionner les modules d'échecs et sélectionnez le module à utiliser.

Modules UCI depuis un serveur distant

DroidFish peut utiliser des modules UCI fonctionnant sur un serveur distant.

  1. Installez un programme d'échecs en réseau sur l'ordinateur distant.
    1. Pour Windows, installez un programme faisant serveur depuis la page Computer Chess Utility.
    2. Pour Linux, mini-inetd depuis le paquet tcputils peut être utilisé.
  2. Sélectionnez Gérer les modules d'échecs dans le menu du programme, créez un nouveau module réseau, et entrez le nom de l'hôte ainsi que le numéro du port pour le module distant.
  3. Depuis le menu du programme, choisissez Gérer les modules d'échecs et Sélectionner les modules d'échecs et sélectionnez le module à utiliser.

Tables de finales Gaviota

Pour utiliser les tables de finales Gaviota:

  1. Copiez les fichiers .gtb.cp4 dans le répertoire DroidFish/gtb de la carte SD.
  2. Utilisez le menu des préférences des Tables de données de finales pour en définir les paramètres d'utilisation.
